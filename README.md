# podcast.mozillabrasil.org.br

Static pages site built with Jekyll.

## Requirements

* ruby
* ruby-devel
* rubygems

## Install

* <code>git clone http://github.com/mozillabrasil/podcast</code>
* <code>cd podcast</code>
* <code>gem install jekyll</code>
* <code>jekyll serve</code>
